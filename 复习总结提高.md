## 基本格式： 题目 + 常见API的复习
[3Sum](https://leetcode.com/problems/3sum/) <br>
```Java
// res = List<List<Integer>>;
List<Integer> single = new ArrayList<>();
single.add(int a);
single.add(int b);
single.add(int c);
res.add(single);
```
可以变换成
```Java
res.add(Arrays.asList(int a, int b, int c));
```

[Count and Say](https://leetcode.com/problems/count-and-say/) <br>
```Java
StringBuilder sb = new StringBuilder();
int value;
sb.append(value);
sb.append((char) (value + '0'))
```
把int转换为char:  1 to '1', 用 (char) (value + '0') <br>
把char 转换为int: '1' to 1, 用 char - '0'
