[Rotate Image](https://leetcode.com/problems/rotate-image/) <br>
1. 环状替换的模块化
```Java
private void rot(int[][] matrix, int n, int row, int col) {
    int initRow = row;
    int initCol = col;
    int prev = matrix[row][col];
    do {
        int nextRow = col;
        int nextCol = n - row - 1;
        int temp = matrix[nextRow][nextCol];
        matrix[nextRow][nextCol] = prev;
        prev = temp;
        row = nextRow;
        col = nextCol;
    } while (row != initRow || col != initCol);
}
```

```Java
    }
}
```

[Maximum Subarray](https://leetcode.com/problems/maximum-subarray/) <br>
线段树的引入。<br>
```Java
private class Segment {
    int iSum, lSum, rSum, mSum;
    private Segment(int iSum, int lSum, int rSum, int mSum) {
        this.iSum = iSum;
        this.lSum = lSum;
        this.rSum = rSum;
        this.mSum = mSum;
    }
}
```
注意区分概念和合并操作的原理： <br>
1. 包含左端点的最大区间和 <br>
2. 包含右端点的最大区间和 <br>
3. 整体区间和 <br>
4. 最大区间和 <br>
